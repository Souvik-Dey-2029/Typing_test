# Typing Speed Test

A simple typing speed test built using HTML, CSS, and JavaScript.

## Features
- Random typing quotes
- Live correct/incorrect highlighting
- Timer in seconds
- WPM (Words Per Minute) calculation

## Tech Used
- HTML
- CSS
- JavaScript

## How to Run
1. Download the project
2. Open `index.html` in a browser
3. Click **Start Game** and begin typing
